Flappy Bird

Application Overview: The user will have to play Flappy Bird... keep the bird up, and don't let it hit anything.

Technology Overview:
1. Programming Language: Python
2. Game Development Library: Pygame
3. Platform: Desktop (Windows, macOS, Linux).
4. Hosting: Self-contained desktop application

Last updated: 11/8/23

Feature List:
1. Bird Flapping: Implement bird movement and gravity.
2. Pipe Obstacles: Create scrolling pipe obstacles.
3. Collision Detection: Check for collisions between the bird and pipes.
4. Scoring System: Keep track of the player's score.
5. Game Over Screen: Display a game over screen with the player's score.
6. Sound Effects: Add sound effects for flapping and collisions.
